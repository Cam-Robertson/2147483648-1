Cookies

The following pages use cookies to store user preferences. No other pages on www.babelstone.co.uk use cookies.


babelmap.html

http://www.babelstone.co.uk/Unicode/babelmap.html stores the following user preferences for BabelMap Online for 31 days:

  • BabelMapFontSize : Font point size
  • BabelMapFontColour : Font colour
  • BabelMapBackColour : Background colour
  • BabelMapHighColour : Highlight colour
  • BabelMapFontMapping : Whether to use default font mapping, custom font mapping, or a specific font
  • BabelMapFontName : Font name to use instead of default or custom mapping, if specified
  • BabelMapBlock : Unicode block to display on loading
  • BabelMapPage : Page of Unicode block to display on loading if the block has multiple pages

unicode.html

http://www.babelstone.co.uk/Unicode/unicode.html stores the following user preferences for the Unicode Slide Show for 31 days:

  • UnicodeSpeed : Character display speed
  • UnicodeFontSize : Font point size
  • UnicodeFontColour : Font colour
  • UnicodeBackColour : Background colour
  • UnicodeCustomFonts : Whether to use custom fonts or not

whatisit.html

http://www.babelstone.co.uk/Unicode/whatisit.html stores the following user preferences for the What Unicode character is this ? tool for 31 days:

  • WhatIsItPrefix : Whether to prefix character names with their code point value or not
